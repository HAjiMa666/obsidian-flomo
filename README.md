# Obsidian Flomo Integration

将 Obsidian 笔记内容快速发送到 Flomo 的插件。

## 功能

- 选中文字右键发送到 Flomo（支持预览和添加标签）
- 右键菜单快速打开 Flomo 输入窗口
- 支持自定义默认标签
- 自动处理标签格式（避免重复 `#`）

## 使用方法

### 配置

1. 在设置中配置 Flomo Webhook URL
2. 可选：设置默认标签（逗号分隔）

### 发送内容

**方式一：选中文字发送**
- 选中文字 → 右键 → "发送到 Flomo"
- 在预览窗口中可添加标签
- 点击发送

**方式二：手动输入**
- 右键 → "打开 Flomo"
- 输入内容和标签
- 点击发送
